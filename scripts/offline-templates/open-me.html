GPK Collection Manager — Offline copy

This folder is a self-contained copy of the viewer. It works with no internet as long as you also load the image backup ZIP once it's open.

What works offline: browsing cards, filtering, puzzle builder, binder view, card details, animations.
What doesn't: connecting your WAX wallet, live NFT ownership, transfers, price alerts. Those need the internet.

Easiest way — double-click the app

  1. Double-click index.html in this folder.
  2. When the app opens, click Offline backup in the top-left, then load your gpk-image-mirror.zip.

If images don't load or the page looks broken, use the local-server option below — some browsers restrict what a page opened from a file can do.

Reliable way — run the tiny local server

Open a terminal in this folder and run one of these one-liners, then open http://localhost:8080:

Windows (needs Python — check with python --version):

python -m http.server 8080

Mac / Linux:

python3 -m http.server 8080

Or with Node.js (any OS):

npx serve . -l 8080

Stop the server with Ctrl+C when you're done.

You also need the image backup ZIP

The viewer is empty on its own — it only ships with the app code. The images live in a separate gpk-image-mirror.zip (~1.2 GB). Grab that from the GitHub Release page while it's still online:

Open the release page

Once you have the image ZIP, keep it next to this folder so you always have both.
